## 3Ô∏è‚É£ Loops

Loops are used when **something needs to repeat**.

### Why loops?

Instead of writing:

```jsx
cout << 1;
cout << 2;
cout << 3;
```

We use a loop:

```jsx
for(int i = 1; i <= 3; i++) {
    cout << i;
}
```

### Types of loops:

- `for` loop ‚Üí when count is known
- `while` loop ‚Üí when condition matters
- `do-while` loop ‚Üí runs at least once
